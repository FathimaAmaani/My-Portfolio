Boutiqa E-commerce Website Prototype

The Boutiqa E-commerce Website Prototype project was undertaken as a UI/UX designer at Brightica Design Agency to develop a high-fidelity prototype for Boutiqa, an online marketplace facilitating transactions between sellers and consumers. The goal was to create a user-centric, rich internet application that ensures an intuitive and engaging user experience for both user types, supporting seller functions like registration, profile management, and product catalogue maintenance, and consumer functions such as registration, product search, detailed product viewing, and checkout. EduLearn: AI-Driven Education Platform (Laravel)

EduLearn, an innovative e-learning platform, sought to enhance student learning by integrating artificial intelligence (AI) to address limitations in traditional online education, such as static content and delayed feedback. The objective was to develop a web application that delivers personalized learning paths, real-time feedback, and intelligent tutoring, making education adaptive, engaging, and accessible. As an Apprentice Systems Developer, I transformed EduLearn by implementing AI-driven features using Laravel and Hugging Face APIs, creating a dynamic platform that tailors content to individual student needs and supports educators with actionable insights.

The enhanced EduLearn platform, built using the Spiral SDLC model, leverages PHP Laravel for backend logic, MySQL for data storage, and Blade templates with HTML, CSS, and JavaScript for a responsive frontend. I integrated Hugging Face’s Mixtral-8x7B API to power three core AI features: personalized learning paths, which analyze student performance to recommend tailored content; real-time feedback, providing instant grading and improvement tips for quizzes; and intelligent tutoring, offering 24/7 chatbot support for student queries. Users log in via secure Breeze authentication, accessing role-specific dashboards (students and teachers). For example, students input quiz answers, and the AI, coded in Laravel, processes them through Hugging Face to deliver immediate feedback and curated learning suggestions, stored in MySQL and displayed on a user-friendly dashboard. Challenges like API token limitations were overcome by switching from OpenAI to Hugging Face’s free-tier API, tested with Postman for reliability, ensuring scalability and cost-efficiency.

Jumpstart Voice-Assistant Web Application (Laravel)

The JumpStart Retail project focuses on addressing the operational challenges of a nationwide fashion retailer with 750 stores and a growing e-commerce presence, which faces increasing complexity in managing customer interactions, supply chain logistics, and marketing analytics. The primary goal was to enhance customer support efficiency and experience by integrating Voice-to-Text AI, a technology that leverages machine learning and natural language processing (NLP) to convert spoken customer queries into text for rapid processing by support systems or AI-driven bots. This solution was developed as a web application using the Laravel framework, with phpMyAdmin and MySQL for secure data management and the Hugging Face API (Mixtral-8x7B) for accurate voice-to-text transcription. The application enables customers to voice queries naturally, integrating seamlessly into platforms like live chat, help desks, or phone systems, thereby reducing response times, enhancing accessibility for users with disabilities, and providing 24/7 support availability. Transcribed data also offers valuable insights into customer needs and trends. A mixed-methods research approach was employed, utilizing quantitative surveys via Google Forms to gauge user preferences and qualitative semi-structured interviews via Microsoft Teams to explore emotional and practical concerns. Findings revealed openness to AI for routine tasks like order tracking but highlighted concerns about accuracy with diverse accents and the need for emotional tone recognition. Secondary research, including case studies from Google Cloud, Amazon Transcribe, and Bank of America’s Erica, confirmed the solution’s viability, demonstrating benefits such as reduced agent workload, faster response times, and improved customer satisfaction. Challenges like multilingual support, speech accuracy, and data privacy were addressed through recommendations for robust AI training with diverse datasets, secure data handling, and hybrid AI-human models to ensure empathy for complex queries. Developed iteratively using Agile methodology, with tools like Visual Studio Code, Cursor AI, and GitHub for version control, the solution ensures scalability, user trust, and alignment with JumpStart’s digital transformation goals, delivering a practical, efficient, and customer-centric approach to modernizing support services.
